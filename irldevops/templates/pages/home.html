Django IRL

A Django Community built on Django

DjangoIRL Stands for DJANGO IN REAL LIFE

This a open project with the goal of building a django community platform running on django by django developers. It is a opportunity for newbie developers to harness skills on a project that is not a throw away or a just a portfolio project.

Technology used
  • Python 2.7
  • Django 1.11
  • Postgresql
  • Bootstrap 4

How to contribute? Just fork https://github.com/brianbola90/DjangoIRL
