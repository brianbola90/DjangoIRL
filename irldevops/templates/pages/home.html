Django IRL

A Django Community built by Django developers

This a Open project aimed at building a django community by building a platform on django by django developers.

How to contribute? Just fork https://github.com/brianbola90/DjangoIRL

This project started as an attempt to build a devops community but it was too broad of a space no cover. One of the key skills a Dev needs is collaboration

This site allows you to work with fellow devs and build something great. All contributions will be added to a contributors page and credit will be given where due.
